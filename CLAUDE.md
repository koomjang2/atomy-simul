# 애터미 직급표 시뮬레이터 - 프로젝트 컨텍스트

## 프로젝트 개요
애터미(Atomy) 네트워크 마케팅의 **직급 달성 최적화 웹 시뮬레이터**.
하위 SM(세일즈마스터)의 몸PV를 날짜별로 최적 배분하여,
하위 SM은 최소 비용으로 직급 달성하고 상위 DM/SRM은 최대 후원수당을 받는
**윈-윈 시나리오 직급표를 자동 생성**하는 웹앱.

## 기술 스택
- Frontend: React + Vite (또는 vanilla HTML/CSS/JS)
- 스타일: Tailwind CSS
- 상태관리: React useState / useReducer
- 빌드: npm run build → 정적 파일 배포 가능

## 핵심 도메인 지식
→ `docs/pv_rules.md` : 바이너리 규칙 + 후원수당 테이블
→ `docs/rank_logic.md` : 전 직급 달성 조건 (SSM~IM)
→ `docs/algorithm.md` : 몸PV 쪼개기 4단계 알고리즘 (핵심)
→ `docs/ui_spec.md` : 화면 구성 및 UX 명세

## 개발 우선순위
1. 핵심 계산 엔진 (algorithm.md의 4단계 로직)
2. 직급표 입력 UI (날짜별 좌/우 PV 입력 + 몸PV)
3. 자동 최적화 버튼
4. 결과 테이블 렌더링 + 수당 계산 출력

## 절대 규칙 (코딩 시 반드시 준수)
- 일요일은 PV 입력 불가 (0으로 잠금)
- 점수를 받은 다음날 좌/우 PV는 0으로 초기화 (소실적)
- DM 이상 직급자는 몸PV 합산으로 직급 달성 불가
- 몸PV는 항상 좌/우 중 더 적은 쪽(소실적)에 합산
- 플래시아웃: 한쪽이 매칭 없이 누적되면 초기화 시 소멸
