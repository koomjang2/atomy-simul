## 주요 화면 컴포넌트

### 1. 조직 트리 패널 (좌측)

- 드래그 앤 드롭 또는 버튼으로 노드 추가
- 각 노드: 이름, 직급(SSM/SM/DM/SRM), 몸PV 풀 입력
- **[+ 좌측 하위 추가] / [+ 우측 하위 추가]** 버튼 각각 구분
  → 어느 라인(좌/우)에 배치되는지 시스템이 정확히 인지해야 함
- 선택된 노드의 직급표가 우측에 표시
- 노드 색상: SM(파란색), DM(주황색), SRM(초록색), STM(보라색)

### 2. 기간 설정 바

```
[연도: 2026▼] [월: 4월▼] [상반기(1~15) ● / 하반기(16~말일) ○]
```

- 선택 즉시 캘린더 자동 생성
- 일요일 자동 감지 및 해당 셀 회색 처리 + 입력 잠금

### 3. 직급표 테이블 (메인)

```
┌────┬────┬──────────┬──────────┬──────────┬──────┬──────────┐
│ 일 │요일│    좌    │    우    │   몸PV   │ 점수 │  일일수당 │
├────┼────┼──────────┼──────────┼──────────┼──────┼──────────┤
│ 16 │ 화 │ [  30  ] │ [  30  ] │          │ 15점 │  6만원   │
│ 17 │ 수 │ [  30  ] │          │          │      │          │
│ 18 │ 목 │          │ [  30  ] │          │ 15점 │  6만원   │ ← 이틀 매칭
│ 19 │ 금 │          │          │          │      │          │
│ 20 │ 토 │ [  30  ] │          │ [  10  ] │      │          │
│ 21 │ 일 │ [회색-잠금]                                       │
│ 22 │ 월 │ [  40  ] │          │ [  20  ] │ 30점 │ 12만원   │
...
├────┴────┼──────────┼──────────┼──────────┼──────┼──────────┤
│  합계   │  250만   │  210만   │  40만    │ 누적 │  총수당  │
│ 달성여부│ ✅ SM 달성                                         │
└─────────┴──────────┴──────────┴──────────┴──────┴──────────┘
```

**셀 규칙:**
- 입력값: 만 단위 숫자 (30 = 30만 PV)
- 점수/수당 열: 자동 계산, 편집 불가 (회색 배경)
- 일요일 행: 전체 회색, 입력 불가
- 이틀 매칭 표시: 색상으로 페어링 (같은 색 = 같은 매칭 쌍)

### 4. 자동 최적화 버튼

```
[🔧 자동 최적화] 클릭 시:
  1. 현재 조직 트리 스캔
  2. algorithm.md의 4단계 실행
  3. 결과를 직급표에 자동 채워넣기
  4. 변경된 셀은 파란 배경으로 하이라이트
```

### 5. 수당 계산 요약 패널 (하단)

```
┌─────────────────────────────────────────┐
│ 📊 수당 계산 요약                        │
│                                         │
│ [SM 홍길동] 직급수당: 약 45만원          │
│   후원수당 합계: 8회 × 6만원 = 48만원   │
│   총 예상 수입: 약 93만원               │
│                                         │
│ [DM 김철수] (상위)                      │
│   최적화 전 수당: 15점×7일 = 630만점    │
│   최적화 후 수당: 30점×3일+15점×4일 =  │
│   → 수당 향상: +약 18만원 예상          │
└─────────────────────────────────────────┘
```

### 6. 내보내기 기능

- [📋 표 복사] : 클립보드 복사 (카카오톡 붙여넣기용)
- [💾 저장] : JSON 형태로 로컬스토리지 저장
- [🖨️ 인쇄] : 프린트 최적화 레이아웃

## 인터랙션 규칙

### 실시간 계산
- 좌/우 PV 입력 시 즉시 점수/수당 재계산
- 합계 행 실시간 업데이트
- 직급 달성 여부 실시간 표시 (초록 ✅ / 빨간 ❌)

### 이틀 매칭 시각화
- 매칭 쌍을 같은 색상으로 하이라이트
- 색상 범례: 빨강/노랑/초록/파랑/하늘/분홍/주황 순환

### 플래시아웃 경고
- 한쪽 PV가 쌓이고 매칭 없이 초기화 위험 시 노란 경고 표시
- "⚠️ 우측 PV 30만 소멸 위험 (다음 매칭 날짜 없음)"

### 몸PV 잔여 표시
- 몸PV 풀 대비 사용량 프로그레스 바
- "몸PV: 20만 / 40만 사용 (잔여 20만)"
