# 애터미 직급 달성 조건

## 기본 원칙
- **달성 기간:** 보름(15일) 단위 → 상반기(1~15일) / 하반기(16일~말일)
- **일요일:** 휴무, PV 연산 배제
- **실제 영업일:** 보통 13일

---

## 직급별 달성 조건

### SSM (세미세일즈마스터 / 세미)
- 좌 150만 PV 이상 + 우 150만 PV 이상 (보름 소실적 기준)
- 몸PV(회원PV) 합산 가능 (더 적은 쪽에 자동 합산)
- 프로모션: 달성 시 20만원(±20%) 지급 (최초 달성 기준 12개월, 최대 12회)

### SM (세일즈마스터 / 판매사)
- 좌 250만 PV 이상 + 우 250만 PV 이상 (보름 소실적 기준)
- 몸PV(회원PV) 합산 가능 (더 적은 쪽에 자동 합산)
- **몸PV 히든카드 예시:** 좌 250만 / 우 210만 / 몸PV 40만
  → 몸PV 40만이 우(소실적)에 합산 → 210+40=250만 → SM 달성

### DM (다이아몬드마스터 / 팀장)
- 좌에 SM 2명 이상 + 우에 SM 2명 이상 (보름 동안 동시 달성)
- **몸PV 직급 달성 불가** (PV 합산으로 DM 달성 안됨)
- 단, 하위 SM의 몸PV는 DM의 일일 후원수당 점수에는 영향을 줌

### SRM (샤론로즈마스터 / 국장)
- 좌에 DM 2명 이상 + 우에 DM 2명 이상 (보름 동안 동시 달성)
- 좌/우 한쪽에만 DM 3명 이상이어도 됨 (양쪽 합산 기준 아님, 좌에 2명 이상 AND 우에 2명 이상)

### STM (스타마스터 / 본부장)
- 좌에 SRM 2명 이상 + 우에 SRM 2명 이상 (보름 동안 동시 달성)
- 필요 하위 인원: SRM 4명, DM 12명, SM 36명 (합계 52명)

### RM (로얄마스터 / 총장)
- 좌에 STM 2명 이상 + 우에 STM 2명 이상
- 필요 하위 인원: STM 4명, SRM 12명, DM 36명, SM 108명 (합계 160명)

### CM (크라운마스터 / 단장)
- 좌에 RM 2명 이상 + 우에 RM 2명 이상
- 필요 하위 인원: RM 4명, STM 12명, SRM 36명, DM 108명, SM 324명 (합계 484명)

### IM (임페리얼마스터 / 임페리얼)
- 좌에 CM 2명 이상 + 우에 CM 2명 이상
- 필요 하위 인원: CM 4명, RM 12명, STM 36명, SRM 108명, DM 324명, SM 972명 (합계 1,456명)

## DM 이상 직급의 달성 구조 이해

```
나 (DM 기준)
├── 좌 (SM1)
│   └── 좌의 좌 (SM2) ← 좌에 SM 2명 달성
└── 우 (SM1)
    └── 우의 우 (SM2) ← 우에 SM 2명 달성
```

- 좌/우 내부에서 같은 라인 안에 SM이 2명이면 됨
- 예: 나의 좌에 SM1이 있고, SM1의 하위에 SM2가 있으면 → 좌 SM 2명 달성
- DM부터 IM까지 같은 원리: 자신과 같은 직급이 좌/우 각각 2명 이상

---

## 몸PV 합산 로직 (SSM/SM 전용)

> ⚠️ **DM 이상 직급자는 몸PV 개념 자체가 없음.**
> 몸PV 입력란은 SSM/SM 노드에만 표시. DM/SRM/STM/RM/CM/IM은 UI에서 몸PV 항목 완전 비표시.

```
if (우PV < 좌PV):
    우PV += 몸PV
elif (좌PV < 우PV):
    좌PV += 몸PV
else:  # 동일할 경우
    우PV += 몸PV  # 기본값: 우에 합산
    
if (좌PV >= target) AND (우PV >= target):
    → 직급 달성
```

- SSM target = 150만
- SM target = 250만

## SSM 프로모션 규칙 (세미세일즈마스터)

- 매월 1~15일 / 16일~말일 소실적 기준 150만 PV 이상
- 해당 기간 SM 이상 미달성자 대상
- 소실적 라인 최소 30만 PV 이상인 경우 몸PV 합산 가능
- 특별 프로모션 20만원(±20%) 지급
- 최초 달성 기준 12개월 한정, 최대 12회
- 미달성 시 추가 12개월 연장 (총 24개월 내 12회 달성)
- 지급 수당 총액이 총 매출의 35% 초과 시 자동 감함
